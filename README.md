# Simple Weather Application

React + Vite weather app using OpenWeatherMap, styled with Tailwind CSS. Supports dark mode and geolocation.

## Setup
1. Install deps
   npm install

2. Create .env.local
   VITE_OPENWEATHER_API_KEY=YOUR_KEY

3. Run dev
   npm run dev

## Scripts
- npm run dev
- npm run build
- npm run preview

## Tech
- React + Vite
- Tailwind CSS (dark mode via class)
- OpenWeatherMap API
